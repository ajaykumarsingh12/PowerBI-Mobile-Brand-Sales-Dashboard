# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Mobile Sales Dashboard** built in **Microsoft Power BI** to analyze mobile sales performance across brands, cities, payment methods, and time periods. The dashboard provides actionable insights through dynamic KPIs and interactive visualizations to support data-driven business decisions.

---


## 📂 Project Files

* `Mobile Sales MobileBrandSales.pbix` – Power BI dashboard
* `Day - 30 - Mobile Sales Data.xlsx` – Sales dataset
* `Images/Dashboard.png` – Dashboard screenshot
* `README.md` – Project documentation

---

## 🛠 Tools Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel

---

## 📈 Dashboard Features

* KPI Cards (Total Sales, Quantity, Transactions, Average Sales)
* City-wise Sales Analysis using Map Visual
* Monthly Sales Trend Analysis
* Customer Ratings Analysis
* Payment Method Distribution
* Mobile Model Performance
* Day-wise Sales Analysis
* Interactive Slicers (Brand, Model, Payment Method, Month, Day)
